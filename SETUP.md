# 🚀 AI-Based Predictive Cloud Load Management - Setup Guide\n\n## 📋 Prerequisites\n\n### Backend Requirements\n- Python 3.9+\n- pip (Python package manager)\n- MongoDB Atlas account (or local MongoDB)\n- Redis (optional, for caching)\n\n### Frontend Requirements\n- Node.js 16+ (LTS recommended)\n- npm or yarn\n\n## 🔧 Installation Steps\n\n### 1. Backend Setup\n\n```bash\n# Navigate to backend directory\ncd backend\n\n# Install Python dependencies\npip install -r requirements.txt\n\n# Set up environment variables\ncp .env.example .env\n# Edit .env file with your configurations\n```\n\n### 2. Environment Configuration\n\nEdit `.env` file:\n\n```env\n# Flask Configuration\nFLASK_ENV=development\nSECRET_KEY=your-super-secret-key-change-in-production\nJWT_SECRET_KEY=your-jwt-secret-key-change-in-production\n\n# MongoDB Configuration\nMONGO_URI=mongodb+srv://username:password@cluster0.mongodb.net/cloud_load_management?retryWrites=true&w=majority\n\n# Email Configuration (optional)\nMAIL_SERVER=smtp.gmail.com\nMAIL_PORT=587\nMAIL_USE_TLS=True\nMAIL_USERNAME=your-email@gmail.com\nMAIL_PASSWORD=your-app-password\n\n# Redis Configuration (optional)\nREDIS_URL=redis://localhost:6379/0\nRATELIMIT_STORAGE_URL=redis://localhost:6379/1\n```\n\n### 3. Database Setup\n\n```bash\n# Train the ML model\npython ml/train_model.py\n\n# Start the Flask application\npython app.py\n```\n\nThe backend will:\n- Create default admin user (admin/admin123)\n- Initialize database collections\n- Load the trained ML model\n- Start on http://localhost:5000\n\n### 4. Frontend Setup\n\n```bash\n# Navigate to frontend directory\ncd frontend/react-app\n\n# Install dependencies\nnpm install\n\n# Start development server\nnpm run dev\n```\n\nFrontend will start on http://localhost:3000\n\n## 🎯 Default Login Credentials\n\n**Admin Account:**\n- Username: `admin`\n- Password: `admin123`\n- Role: Administrator\n\n**Create User Account:**\n- Use the registration Backend Deployment (Production)\n\n1. **Environment Setup:**\n   ```bash\n   # Set production environment variables\n   export FLASK_ENV=production\n   export SECRET_KEY=your-production-secret-key\n   export MONGO_URI=your-production-mongodb-uri\n   ```\n\n2. **Using Gunicorn:**\n   ```bash\n   pip install gunicorn\n   gunicorn -w 4 -b 0.0.0.0:5000 app:app\n   ```\n\n3. **Using Docker:**\n   ```dockerfile\n   FROM python:3.9-slim\n   WORKDIR /app\n   COPY requirements.txt .\n   RUN pip install -r requirements.txt\n   COPY . .\n   EXPOSE 5000\n   CMD [\"gunicorn\", \"-w\", \"4\", \"-b\", \"0.0.0.0:5000\", \"app:app\"]\n   ```\n\n### Frontend Deployment\n\n1. **Build for Production:**\n   ```bash\n   npm run build\n   ```\n\n2. **Deploy to Static Hosting:**\n   - Netlify\n   - Vercel\n
